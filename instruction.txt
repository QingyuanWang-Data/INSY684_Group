INSY684 Home Credit Risk MLOps Team - Final Review Guide

Recommended Windows demo:
1. Install uv: https://docs.astral.sh/uv/getting-started/installation/
2. Double-click run_dashboard.cmd for the interactive policy dashboard.
3. Double-click run_api.cmd for the model prediction API.
4. Open http://localhost:8000/docs for the API contract.

Cross-platform commands:
- uv sync --all-groups --locked
- uv run --group dashboard streamlit run INSY684_Group-Extend/INSY684_Group-Extend/monitoring_dashboard/monitoring_dashboard.py
- uv run uvicorn homecredit_service.main:app --host 0.0.0.0 --port 8000
- uv run pytest

The approved model bundle and dashboard evidence are included. Raw Kaggle CSV
files are required only for retraining. Full documentation is in README.md and
docs/SUBMISSION_INFO.md.
